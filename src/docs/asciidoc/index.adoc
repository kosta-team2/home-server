== 개요
'Kosta-Team2 Final Project' 의 백엔드 API 문서입니다.

이 문서는 실거래가 데이터, 아파트 단지 정보, 회원/인증 등
Kosta-Team2에서 제공하는 모든 REST API의 명세를 제공합니다.

=== 응답형식
본 프로젝트는 성공 응답에 대해 별도의 공통 래퍼(JSON)를 사용하지 않습니다. +
HTTP 상태 코드 + 실제 리소스 JSON 을 그대로 반환합니다.

|====
|상황 | HTTP 상태 코드 | 응답 Body 예시

| 조회 성공 (데이터 있음)
| `200 OK`
a|[source,json]
----
{
  "id": 1,
  "name": "kosta-apartment",
  "address": "서울시 ..."
}
----

| 생성 성공
| `201 Created`
a|[source,json]
----
{
  "id": 10,
  "name": "new-resource"
}
----

| 삭제 성공 (응답 본문 없음)
| `204 No Content`
a|[source,json]
|====
